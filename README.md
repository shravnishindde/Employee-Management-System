# Employee Management System

## Features
- Employee Login
- Admin Dashboard
- Attendance Management
- Leave Management
- Payroll Generation
- Reports and Graphs

## Technologies Used
- Java
- MySQL
- JDBC
- Servlet
- Swing / JSP (according to your project)

## How to Run
1. Import project into Eclipse/NetBeans
2. Configure MySQL database
3. Run the main file

## Project Objective
The Employee Management System helps organizations manage employee records, attendance, leaves, and payroll efficiently through an automated system.
